# 📖 คู่มือการใช้งานระบบสืบค้นศัพท์สัตวแพทย์มาตรฐาน 
## (KAHIS SA-PDT & SNOMED CT Veterinary Extension User Guide)

Welcome to **KAHIS Terminology System** — ระบบฐานข้อมูลและสืบค้นรหัสมาตรฐานสัตวแพทย์สำหรับระบบเวชระเบียนอิเล็กทรอนิกส์โรงพยาบาลสัตว์ ม.เกษตร (ประเทศไทย)

---

### 📌 1. บทนำและวัตถุประสงค์ (Introduction & Purpose)

ระบบ **KAHIS Terminology** พัฒนาขึ้นเพื่อเป็นศูนย์กลางการสืบค้น และจับคู่รหัสมาตรฐานคำศัพท์สัตวแพทย์ โดยเชื่อมโยงรหัสมาตรฐาน 3 แหล่งอ้างอิงเข้าด้วยกัน ได้แก่:
1. **SA-PDT (Small Animal Problems and Diagnoses Terms)**: มาตรฐานคำศัพท์สัตวแพทย์อ้างอิงจาก [Virginia-Maryland College of Veterinary Medicine (VTSL)](https://vtsl.vetmed.vt.edu/sa/)
2. **VSCT (Veterinary Extension to SNOMED CT)**: มาตรฐานคำศัพท์สัตวแพทย์สากล อ้างอิงจากแหล่งเดียวกันคือ Virginia-Maryland College of Veterinary Medicine
3. **SCT International (SNOMED CT International Core)**: มาตรฐานคำศัพท์การแพทย์สากลระดับโลก

**วัตถุประสงค์หลัก**:
- เพื่อให้การบันทึกประวัติการรักษา (EHR/EMR) ในโรงพยาบาลสัตว์ ม.เกษตร มีมาตรฐานเดียวกัน
- เพื่อพัฒนาการสร้างฐานข้อมูลการสืบค้นเฉพาะแห่งสำหรับโรงพยาบาลสัตว์ ม.เกษตร (ประเทศไทย)
- เพื่อรองรับการสืบค้นด้วยภาษาไทย ภาษาอังกฤษ คำย่อทางการแพทย์ หรือรหัส Concept ID
- เพื่อเชื่อมโยงข้อมูลสุขภาพสัตว์ไทยเข้าสู่มาตรฐานสากลระดับโลก

### 💡 2. หลักการทำงานของระบบ (Core System Principles)

#### 2.1 โครงสร้างแบบมโนทัศน์ (Concept-Based Architecture)
ระบบจัดเก็บข้อมูลแบบ "มโนทัศน์" (Concept) ซึ่งความหมาย 1 มโนทัศน์ (Concept ID) สามารถมีคำเรียกได้หลายรูปแบบ (Synonyms & Aliases) ทั้งภาษาไทย ภาษาอังกฤษ และคำย่อ โดยเมื่อสืบค้นด้วยคำใดๆ ระบบจะดึงมโนทัศน์ที่ถูกต้องขึ้นมาแสดงผลเสมอ

#### 2.2 ระบบสีป้ายกำกับมาตรฐาน (Unified Badge Design System Tokens)
เพื่อให้อ่านข้อมูลได้ง่าย ชัดเจน และสบายตา ระบบได้กำหนดโค้ดสีกลางสำหรับแหล่งข้อมูลดังนี้:

| ป้ายกำกับ | สีพื้นหลัง | สีตัวอักษร | ความหมาย |
| :---: | :---: | :---: | :--- |
| **`SA-PDT`** | 🔵 สีฟ้าอ่อน (`#dbeafe`) | สีน้ำเงินเข้ม (`#1e40af`) | ข้อมูลมาจากฐานข้อมูล SA-PDT |
| **`VSCT`** | 🟢 สีเขียวอ่อน (`#d1fae5`) | สีเขียวเข้ม (`#065f46`) | ข้อมูลมาจากฐานข้อมูล VSCT |
| **`VSCT ONLY`** | 🟡 สีเหลืองมัสตาร์ด (`#fef3c7`) | สีน้ำตาลเข้ม (`#92400e`) | มโนทัศน์ที่มีเฉพาะในฐานข้อมูล VSCT (ไม่มีใน SA-PDT) |
| **`SCT-INTER`** | ⚪ สีเทาสเลท (`#f1f5f9`) | สีเทาเข้ม (`#475569`) | มโนทัศน์ขยายมาจาก SNOMED CT สากล |
| **`ku`** | 🩵 สีเขียวน้ำทะเล (`#ccfbf1`) | สีเขียวเข้ม (`#115e59`) | คำพ้องภาษาไทย/ภาษาอังกฤษที่เพิ่มโดยโครงการ KU |
| **`SYSTEM`** | 🟣 สีม่วงลาเวนเดอร์ (`#f3e8ff`) | สีม่วงเข้ม (`#6b21a8`) | ข้อมูลระบบและเมทาเดทา |

### 🔍 3. วิธีการใช้งานระบบสืบค้น (Step-by-Step User Guide)

#### ขั้นตอนที่ 1: การป้อนคำค้นหา (Search Execution)
ในช่องค้นหาด้านซ้ายมือ คุณสามารถพิมพ์ค้นหาได้หลายรูปแบบ:
- **ค้นหาด้วยภาษาไทย**: เช่น `ท้องเสีย`, `ไอ`, `มีน้ำมูก`, `หัวใจโต`, `นิ่วในไต`, `ขาเจ็บ`, `อ้วก`
- **ค้นหาด้วยภาษาอังกฤษ**: เช่น `Diarrhea`, `Cough`, `Cardiomegaly`, `Renal calculi`, `Pyometra`
- **ค้นหาด้วยคำย่อทางการแพทย์**: เช่น `FLUTD`, `IVDD`, `CCLR`, `TCC`, `SCC`, `CRF`, `CKD`
- **ค้นหาด้วยรหัส Concept ID**: เช่น `95570007`, `70650003`, `352621000009109`

*(ระบบรองรับการค้นหาแบบค้นคำขึ้นต้นหลายคำ Prefix Search เช่น พิมพ์ `shou frac` เพื่อหา Shoulder fracture ได้)*

#### ขั้นตอนที่ 2: การใช้งานตัวกรอง (Filtering Options)
เมนูด้านซ้ายช่วยให้คุณบีบผลการค้นหาให้แคบลงได้:
- **Status (สถานะ)**: เลือกเฉพาะมโนทัศน์ที่ยังใช้งานอยู่ (`Active`) หรือรวมมโนทัศน์ที่ยกเลิกแล้ว (`Inactive`)
- **Semantic Tags (หมวดหมู่)**: กรองตามประเภททางคลินิก เช่น `disorder` (โรค/ความผิดปกติ), `finding` (อาการ/ผลตรวจ), `procedure` (หัตถการ/การรักษา), `organism` (เชื้อโรค/สิ่งมีชีวิต)

#### ขั้นตอนที่ 3: การดูผลการค้นหาและการจัดเรียง (Results List)
- ตารางฝั่งซ้ายจะแสดงชื่อภาษาอังกฤษหลัก พร้อมรหัส และป้ายกำกับแหล่งข้อมูล (`SA-PDT`, `VSCT`, `SCT-Inter Only`) ในบรรทัดใหม่อย่างเป็นระเบียบ
- คลิกที่แถบรายการเพื่อดูรายละเอียดมโนทัศน์ฉบับเต็มในตารางฝั่งขวา

#### ขั้นตอนที่ 4: การใช้งานตารางรายละเอียดมโนทัศน์ (Concept Details Panel)
การ์ดฝั่งขวาจะแสดงข้อมูลเชิงลึกของมโนทัศน์ที่เลือก:
- **Header Box**: แสดงรหัส Concept ID, ชื่อ FSN หลัก, สถานะ, และป้ายกำกับแหล่งข้อมูล
- **Synonyms & Aliases**: แสดงคำพ้องทั้งหมด ทั้งภาษาไทย (ป้าย `[ku]`), ภาษาอังกฤษ (ป้าย `[en]`) และ FSN
- **Attributes & Hierarchy**: แสดงระบบอวัยวะ (`Body System`), มโนทัศน์แม่ (`Parents`) และมโนทัศน์ลูก (`Children`) ในโครงสร้างต้นไม้

#### ขั้นตอนที่ 5: การเพิ่มคำพ้องประจำโครงการ (Add KU Custom Synonym)
หากสัตวแพทย์ต้องการเสนอเพิ่มคำพ้องภาษาไทยใหม่ในระบบ:
1. คลิกปุ่ม **`[+ เพิ่มคำพ้อง / Add Synonym]`** ในการ์ดรายละเอียดมโนทัศน์
2. พิมพ์คำพ้องภาษาไทยหรือคำย่อที่ต้องการ
3. กรอกรหัส PIN สิทธิ์การบันทึก แล้วกด **`[💾 บันทึกคำพ้อง]`** ข้อมูลจะถูกบันทึกลงระบบทันที

### 📚 4. อภิธานศัพท์และคำย่อทางการแพทย์สัตวแพทย์ (Glossary & Abbreviations)

#### 4.1 คำย่อเกี่ยวกับระบบและมาตรฐาน (System & Standard Terms)

| คำย่อ | คำเต็มภาษาอังกฤษ | ความหมาย / คำอธิบาย |
| :---: | :--- | :--- |
| **KAHIS** | Kasetsart Animal Hospital Information System | ระบบสารสนเทศโรงพยาบาลสัตว์ มหาวิทยาลัยเกษตรศาสตร์ |
| **SA-PDT** | Small Animal Problems and Diagnoses Terms | รหัสมาตรฐานคำศัพท์สัตวแพทย์จาก Virginia-Maryland College of Veterinary Medicine |
| **VSCT** | Veterinary Extension to SNOMED CT | ส่วนขยายคำศัพท์สัตวแพทย์สากลของ SNOMED CT จาก Virginia-Maryland College of Veterinary Medicine |
| **SNOMED CT** | Systematized Nomenclature of Medicine -- Clinical Terms | มาตรฐานคำศัพท์ทางการแพทย์และการสัตวแพทย์สากลระดับโลก |
| **FSN** | Fully Specified Name | ชื่อมโนทัศน์ฉบับเต็มที่ระบุหมวดหมู่ในวงเล็บ เช่น `Cough (finding)` |
| **PT** | Preferred Term | ชื่อเรียกหลักที่เป็นที่นิยมใช้มากที่สุดในมโนทัศน์นั้นๆ |
| **ku** | Kasetsart University Custom Synonym | คำพ้องประจำโครงการที่เสนอแนะและบันทึกโดยทีมสัตวแพทย์ มก. |
| **EHR / EMR** | Electronic Health / Medical Record | ระบบบันทึกประวัติการรักษาและเวชระเบียนอิเล็กทรอนิกส์ |

#### 4.2 คำย่อโรคและอาการคลินิกสัตวแพทย์ที่ใช้บ่อย (Common Veterinary Clinical Abbreviations)

| คำย่อ | ชื่อเต็มภาษาอังกฤษ | คำอธิบาย / ชื่อภาษาไทย |
| :---: | :--- | :--- |
| **FLUTD** | Feline Lower Urinary Tract Disease | กลุ่มโรคทางเดินปัสสาวะส่วนล่างในแมว (ปัสสาวะขัด/เบ่ง/เป็นเลือด) |
| **IVDD** | Intervertebral Disc Disease | โรคหมอนรองกระดูกทับเส้นประสาท (ทำให้เกิดอาการปวด/อัมพาต) |
| **CCLR** | Cranial Cruciate Ligament Rupture | ภาวะเอ็นไขว้หน้าข้อเข่าขาด (สาเหตุหลักของอาการขาเจ็บในสุนัข) |
| **CKD** | Chronic Kidney Disease | โรคไตเรื้อรัง (ภาวะการทำงานของไตเสื่อมลงแบบถาวร) |
| **CRF** | Chronic Renal Failure | ภาวะไตวายเรื้อรัง |
| **AKI** | Acute Kidney Injury | ภาวะไตวายเฉียบพลัน |
| **GDV** | Gastric Dilatation-Volvulus | ภาวะกระเพาะอาหารขยายและบิดหมุน (ฉุกเฉินในสุนัขพันธุ์ใหญ่) |
| **UTI** | Urinary Tract Infection | การติดเชื้อในระบบทางเดินปัสสาวะ |
| **TCC** | Transitional Cell Carcinoma | มะเร็งเยื่อบุผิวชนิดทรานซิชันนัล (พบมากในกระเพาะปัสสาวะ/ท่อไต) |
| **SCC** | Squamous Cell Carcinoma | มะเร็งเซลล์ผิวหนังชนิดสแควโมส |
| **MCT** | Mast Cell Tumor | มะเร็งแมสต์เซลล์ (เนื้องอกผิวหนังที่พบได้บ่อยในสุนัข) |
| **OSA** | Osteosarcoma | มะเร็งกระดูก |
| **HSA** | Hemangiosarcoma | มะเร็งหลอดเลือด (พบมากที่ม้ามและหัวใจ) |
| **KCS** | Keratoconjunctivitis Sicca | โรคตาแห้งในสัตว์เลี้ยง (ภาวะน้ำตาแห้ง) |
